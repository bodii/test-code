### 面试题 16.01. Swap Numbers LCCI
Write a function to swap a number in place (that is, without temporary variables).

Example:

	Input: numbers = [1,2]
	Output: [2,1]

Note:

* numbers.length == 2
* -2147483647 <= numbers[i] <= 2147483647
