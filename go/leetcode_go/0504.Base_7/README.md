### 504. Base 7
Given an integer num, return a string of its base 7 representation.



Example 1:

	Input: num = 100
	Output: "202"

Example 2:

	Input: num = -7
	Output: "-10"



Constraints:

* -10^7 <= num <= 10^7

### 504. 七进制数
给定一个整数 num，将其转化为 7 进制，并以字符串形式输出。



示例 1:

	输入: num = 100
	输出: "202"

示例 2:

	输入: num = -7
	输出: "-10"



提示：

* -10^7 <= num <= 10^7
